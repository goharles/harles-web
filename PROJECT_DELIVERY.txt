================================================================================
CUSTOMIZATION GUIDE
================================================================================

UPDATE COMPANY INFO
- Edit component files in src/components/
- Modify data arrays (services, testimonials, FAQs)
- Update navigation links

CHANGE COLORS
- Edit tailwind.config.js
- Update CSS variables in globals.css

ADD NEW PAGES
- Create new folder in src/app/
- Add page.tsx file
- Import Header and Footer

UPDATE CONTENT
- Edit text in component files
- Modify arrays for dynamic content
- Update links and navigation
